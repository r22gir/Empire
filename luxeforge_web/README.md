# LuxeForge Web

Marketing website for LuxeForge — the AI-powered software platform for custom workroom businesses.

## Features

- 🏠 **Homepage** — Marketing landing page with hero, features overview, and CTA
- 💰 **Pricing** — Three tiers: Solo ($79/mo), Pro ($249/mo), Enterprise ($599/mo)
- ✨ **Features** — AI chat intake, photo measurements, auto-quotes, production queue
- 📬 **Contact** — Demo request form

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/r22gir/Empire/tree/main/luxeforge_web)

## Environment Variables

Create a `.env.local` file (see `.env.example`):

```
NEXT_PUBLIC_SITE_URL=https://luxeforge.com
```

## Pricing

| Plan | Price | Best For |
|------|-------|----------|
| Solo | $79/mo | Independent workrooms, 1-2 users |
| Pro | $249/mo | Growing studios, up to 10 users |
| Enterprise | $599/mo | Large operations, unlimited users |

## LuxeForge Pricing Model

- Drapery: $95–150/width
- Roman shades: $15–20/sqft
- Fabric markup: 30–50%
- Hardware markup: 50%
- Installation: $45–195/window
- Payment terms: 50% deposit
