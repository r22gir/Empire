# RelistApp 7-Day Launch Plan
## Drop-Ship Arbitrage Sprint: 0 to 800 Listings

*Created: April 2, 2026*

---

## Day 1: Foundation Setup
**Time Required: 6-8 hours**

### Account Setup
- [ ] Create/verify eBay seller account (Business account preferred)
  - Enable Managed Payments
  - Set up eBay Store subscription (Basic: ~$7.95/mo for 250 free listings)
  - Configure return policy (30-day returns, buyer pays return shipping)
  - Set up shipping profiles (free shipping built into price)
- [ ] Create Mercari seller account
  - Complete identity verification
  - Set up direct deposit for payouts
- [ ] Create Facebook Marketplace seller profile
  - Enable shipping (note: FB removed prepaid labels Feb 2025, use PirateShip)
  - Join local buy/sell groups for local pickup items
- [ ] Create Poshmark seller account
  - Complete closet setup with profile photo and bio
  - Share a few items to activate algorithm

### RelistApp Configuration
- [ ] Connect all marketplace accounts to RelistApp
- [ ] Configure profit margin thresholds (minimum 25% after all fees)
- [ ] Set up fee calculators per platform:
  - eBay: 12-15% FVF (includes payment processing) + $0.30-0.40/order
  - Mercari: 10% flat
  - FB Marketplace: 10% shipped, 0% local
  - Poshmark: 20% (over $15) or $2.95 flat (under $15)
- [ ] Set up supplier source accounts (wholesale, not retail arbitrage)
- [ ] Configure notification alerts for sales

### Product Research (Target: 50 Products Identified)
- [ ] Scan Amazon Movers & Shakers for trending items
- [ ] Check Google Trends for rising product searches
- [ ] Identify 50 products with minimum 30% margin after ALL fees
- [ ] Create spreadsheet: source price, sell price, fees per platform, net profit

### What to Watch For
- Account verification delays (can take 24-48 hours)
- eBay selling limits for new accounts (typically 10 items / $500 first month -- request increase immediately)
- Mercari requires phone verification
- FB Marketplace shipping availability varies by region

## Day 2: First 50 Listings Live
**Time Required: 6-8 hours**

### Listing Creation (Target: 50 live, identify next 50)
- [ ] List first 50 products on eBay (primary platform)
  - Use item specifics fully (eBay rewards complete listings)
  - Include 5+ photos per listing (use supplier photos initially)
  - Write keyword-rich titles (80 chars max, front-load keywords)
  - Set competitive pricing (check sold comps, price 5-10% below)
- [ ] Cross-list top 20 highest-margin items to Mercari
- [ ] Cross-list top 10 to FB Marketplace

### Product Research Expansion (Target: 100 products identified)
- [ ] Scan categories: Home & Kitchen, Beauty, Electronics accessories, Pet supplies
- [ ] Use eBay Terapeak / sold listings to validate demand
- [ ] Identify 50 more products meeting margin thresholds
- [ ] Prioritize lightweight items (lower shipping = higher margin)

### Supplier Verification
- [ ] Confirm 2-3 wholesale suppliers with reliable shipping (2-5 day delivery)
- [ ] Test order 3-5 items to verify quality and shipping speed
- [ ] Document supplier contact info, order process, return policies

### What to Watch For
- eBay listing quality score (avoid duplicate listings, use unique descriptions)
- Mercari listing approval time
- Supplier shipping times -- anything over 5 days will kill your metrics
- Price fluctuations on source products

## Day 3: Scale to 150 + Multi-Platform Push
**Time Required: 8-10 hours**

### Listing Expansion (Target: 150 total across platforms)
- [ ] List 50 new products on eBay (total: 100 on eBay)
- [ ] Cross-list all eBay items to Mercari (total: 100 on Mercari)
- [ ] List 50 items on FB Marketplace with shipping
- [ ] List fashion/apparel items on Poshmark (if applicable)

### Platform-Specific Optimization
- [ ] eBay: Enable Promoted Listings on top 20 items (start at 2% ad rate)
- [ ] Mercari: Promote top 10 items, share listings to followers
- [ ] FB Marketplace: Boost 5 listings ($1-2/day each)
- [ ] Poshmark: Share closet items, follow potential buyers

### Inventory Management
- [ ] Set up inventory tracking in RelistApp (prevent overselling)
- [ ] Link listings across platforms (auto-deactivate when sold)
- [ ] Create pricing rules: eBay base price, Mercari +5%, Poshmark +10% (higher fees)

### What to Watch For
- Overselling risk (item sells on two platforms simultaneously)
- eBay defect rate -- respond to all messages within 24 hours
- Mercari rating requests -- ship fast, get 5-star reviews early
- FB Marketplace scam messages (ignore "is this still available" bots)

## Day 4: First Sales + Order Processing
**Time Required: 4-6 hours (plus order processing as sales come in)**

### Order Fulfillment (Target: first 10-20 sales)
- [ ] Process each sale immediately:
  1. Receive sale notification
  2. Place order with wholesale supplier
  3. Enter tracking number on selling platform
  4. Send buyer a thank-you message
- [ ] Track order-to-delivery time for each supplier
- [ ] Document any issues (wrong item, slow shipping, quality problems)

### Listing Expansion (Target: 250 total)
- [ ] Add 100 new listings across platforms
- [ ] Focus on categories generating views/watchers from Days 1-3
- [ ] Test new categories: pet supplies, phone accessories, tools
- [ ] Begin listing seasonal items (spring/summer products for April)

### Analytics Review
- [ ] Check view counts per listing -- identify top performers
- [ ] Review click-through rates on eBay Promoted Listings
- [ ] Compare performance across platforms (which platform converts best?)
- [ ] Calculate actual profit per sale (include all fees + shipping)

### What to Watch For
- Supplier out-of-stock situations -- have backup suppliers ready
- Shipping time complaints -- communicate proactively with buyers
- Payment hold periods (eBay holds funds for new sellers up to 21 days)
- Mercari 3-day auto-rating window

## Day 5: Optimize and Double Down
**Time Required: 6-8 hours**

### Performance Audit
- [ ] Kill zero-view listings after 72 hours (re-list with new titles/photos)
- [ ] Identify top 10% of listings by views -- create similar listings
- [ ] Remove any product with margin under 20% after real fee calculation
- [ ] Update pricing on slow movers (10-15% price reduction)

### Winner Expansion
- [ ] Double down on winning categories:
  - List 10 variations of each top seller (different colors, sizes, bundles)
  - Cross-list ALL winners to ALL platforms
  - Increase Promoted Listings budget on winners (3-5% ad rate)
- [ ] Create bundles from related products (higher AOV, same shipping)

### Listing Target: 400 Total
- [ ] Add 150 new listings focused on proven categories
- [ ] Refresh Day 1-2 listings with better photos and descriptions
- [ ] A/B test pricing on duplicate listings (different platforms)

### Process Documentation
- [ ] Document the full order-to-delivery workflow
- [ ] Create supplier comparison chart (speed, reliability, pricing)
- [ ] Calculate true hourly rate (revenue - costs / hours worked)

### What to Watch For
- eBay Best Match algorithm -- sales velocity matters, so price to sell
- Mercari algorithm favors fresh listings and active sellers
- Customer messages about shipping times -- set expectations in listing
- Return requests -- handle gracefully to protect seller metrics

### Scaling Infrastructure

- [ ] Configure shipping notification templates

### Listing Target: 600 Total

- [ ] Use RelistApp Bulk Cross-Lister for one-click multi-platform posting

## Critical Success Factors

1. **Speed to listing**: More listings = more visibility = more sales. Volume is king in arbitrage.
2. **Margin discipline**: Never list below 25% net margin. Kill losers fast.
3. **Supplier reliability**: One bad supplier ruins your metrics. Test before scaling.
4. **Multi-platform presence**: Same product on 4 platforms = 4x the eyeballs.
5. **Policy compliance**: Use wholesale suppliers, NOT retail arbitrage from Amazon/Walmart (eBay will ban you).
6. **Customer service**: Fast shipping, fast responses, easy returns = algorithm boost on every platform.
