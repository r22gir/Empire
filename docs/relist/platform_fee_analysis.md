# RelistApp Platform Fee Analysis (April 2026)

> Research-backed fee breakdown for every platform in the drop-ship arbitrage chain.
> All figures reflect 2026 published rates unless noted.

---

## SELLING PLATFORMS (Where You List & Sell)

---

### 1. eBay

| Fee Type | Amount |
|---|---|
| Listing fee | Free (first 250/month without store) |
| Final value fee | 13.25% most categories (no store); ~12.35% with Basic Store |
| Per-order fee | $0.30 (orders <= $10); $0.40 (orders > $10) |
| Payment processing | Included in final value fee |
| Shipping labels | USPS Ground Advantage via eBay: ~$5.09-$6.42 (under 1 lb, commercial rate, Zone 1) |
| Monthly subscription | $0 (no store), $7.95/mo (Starter), $21.95/mo (Basic), $59.95/mo (Premium) |
| **Effective fee on $15 item** | $15 x 13.25% + $0.40 = **$2.39 (15.9%)** |
| **Effective fee on $30 item** | $30 x 13.25% + $0.40 = **$4.38 (14.6%)** |

Notes:
- Final value fee applies to item price + shipping charged to buyer + sales tax.
- Basic Store ($21.95/mo) reduces FVF by ~0.5-1% -- worthwhile above ~50 sales/month.
- As of April 26, 2026: USPS rates increase 8% (fuel surcharge).

---

### 2. Mercari

| Fee Type | Amount |
|---|---|
| Listing fee | Free |
| Selling fee | 10% of (item price + buyer-paid shipping) |
| Payment processing | Included (no separate charge since Jan 2025 restructure) |
| Shipping labels | Prepaid label options: ~$5.25 (up to 1 lb), ~$8.40 (1-3 lb) |
| Monthly subscription | None |
| Direct deposit | Free (ACH); $2 fee for rejected transactions |
| **Effective fee on $15 item** | $15 x 10% = **$1.50 (10.0%)** |
| **Effective fee on $30 item** | $30 x 10% = **$3.00 (10.0%)** |

Notes:
- Mercari went from 0% seller fee (2024) back to 10% (Jan 2025 onward).
- 10% is calculated on item price + any buyer-paid shipping.
- Buyer also pays a 3.6% protection fee (does not affect seller).

---

### 3. Poshmark

| Fee Type | Amount |
|---|---|
| Listing fee | Free |
| Commission (< $15) | Flat $2.95 |
| Commission (>= $15) | 20% of sale price |
| Payment processing | Included in commission |
| Shipping | Buyer pays flat $7.67 (prepaid USPS Priority Mail label included) |
| Monthly subscription | None |
| **Effective fee on $15 item** | $15 x 20% = **$3.00 (20.0%)** |
| **Effective fee on $30 item** | $30 x 20% = **$6.00 (20.0%)** |

Notes:
- Poshmark's 20% is the highest commission among major platforms.
- The $2.95 flat fee on items under $15 = 19.7% on a $15 item, so it converges.
- Shipping is fully covered for buyer -- no seller shipping cost. This is a major advantage.
- Best suited for fashion/apparel where buyers expect Poshmark and margins are higher.

---

### 4. Facebook Marketplace

| Fee Type | Amount |
|---|---|
| Listing fee | Free |
| Selling fee (shipped) | 10% of total (item + shipping + tax); min $0.80 |
| Selling fee (local pickup) | **$0 -- completely free** |
| Payment processing | Included in selling fee |
| Shipping labels | Seller provides own label (prepaid labels discontinued Feb 2025) |
| Monthly subscription | None |
| **Effective fee on $15 item (shipped)** | $15 x 10% = **$1.50 (10.0%)** |
| **Effective fee on $30 item (shipped)** | $30 x 10% = **$3.00 (10.0%)** |
| **Effective fee on $15 item (local)** | **$0.00 (0.0%)** |

Notes:
- FB Marketplace is the lowest-fee shipped platform at 10% (tied with Mercari).
- Local pickup = zero fees -- best channel for high-volume, low-margin items if you can handle local logistics.
- Some sources still cite the old 5% rate; the current rate for shipped items is 10%.
- No prepaid shipping labels since Feb 2025 -- you must purchase your own.

---

### 5. Amazon (as a Seller)

| Fee Type | Amount |
|---|---|
| Listing fee | Free |
| Per-item fee (Individual plan) | $0.99/item sold |
| Monthly subscription (Professional) | $39.99/month (no per-item fee) |
| Referral fee | 8-15% (most categories 15%) |
| Payment processing | Included in referral fee |
| FBA fulfillment (small standard, <1 lb) | ~$3.22-$3.86/unit |
| FBM shipping | Seller-arranged (USPS/UPS rates) |
| **Effective fee on $15 item (FBM, Pro)** | $15 x 15% = **$2.25 (15.0%) + shipping** |
| **Effective fee on $30 item (FBM, Pro)** | $30 x 15% = **$4.50 (15.0%) + shipping** |
| **Effective fee on $15 item (FBA, Pro)** | $2.25 referral + ~$3.50 FBA = **$5.75 (38.3%)** |

Notes:
- Professional plan ($39.99/mo) breaks even at 40+ items/month vs Individual ($0.99/item).
- FBA adds $3-6/unit fulfillment on top of the 15% referral fee -- very expensive for low-price items.
- Amazon is primarily a SOURCE platform for arbitrage, not a selling destination for low-margin items.
- Referral fees by key categories: Electronics 8%, Home 15%, Clothing 17%, Beauty 8% (items >$10).

---

### 6. Etsy

| Fee Type | Amount |
|---|---|
| Listing fee | $0.20 per listing (auto-renews on sale) |
| Transaction fee | 6.5% of (item price + shipping) |
| Payment processing | 3% + $0.25 per transaction |
| Monthly subscription | None (free), or Etsy Plus $10/mo for extra features |
| Shipping labels | USPS discounted rates available |
| **Effective fee on $15 item** | $0.20 + ($15 x 6.5%) + ($15 x 3% + $0.25) = $0.20 + $0.98 + $0.70 = **$1.88 (12.5%)** |
| **Effective fee on $30 item** | $0.20 + ($30 x 6.5%) + ($30 x 3% + $0.25) = $0.20 + $1.95 + $1.15 = **$3.30 (11.0%)** |

Notes:
- Total effective rate is ~9.5% + $0.45 fixed per sale (listing + processing flat).
- Etsy Ads (optional) adds 12-15% of ad-attributed sale if used.
- Etsy is best for handmade, vintage, and craft supplies -- limited arbitrage potential for mass-market goods.
- Offsite Ads fee: 15% (or 12% for shops >$10K annual sales) on sales driven by Etsy's offsite ads. This is mandatory and can significantly eat margins.

---

### 7. TikTok Shop

| Fee Type | Amount |
|---|---|
| Listing fee | Free |
| Referral/commission fee | 6% baseline (varies 2-8% by category) |
| Payment processing | ~1% additional |
| New seller promo | 1.8% commission on first $100K in sales (first 90 days) |
| Shipping | Seller-arranged or TikTok Shipping (rates vary) |
| Monthly subscription | None |
| **Effective fee on $15 item** | $15 x 7% = **$1.05 (7.0%)** |
| **Effective fee on $30 item** | $30 x 7% = **$2.10 (7.0%)** |
| **New seller promo on $15 item** | $15 x 2.8% = **$0.42 (2.8%)** |

Notes:
- TikTok Shop has the lowest platform fees in the ecosystem at ~7% all-in.
- New seller promo (1.8% + 1% processing = 2.8%) is exceptional for first 90 days / $100K.
- WARNING: Affiliate commissions (if using creators) average 13% additional -- bringing total to 20%+.
- Content-driven platform; requires video/engagement, not just listing.
- Category rates: Fashion 8%, Electronics 3%, Beauty 5%, Home 5%.

---

### 8. Walmart Marketplace

| Fee Type | Amount |
|---|---|
| Listing fee | Free |
| Referral fee | 6-15% by category (most categories 15%) |
| Payment processing | Included in referral fee |
| Monthly subscription | **None -- $0/month** |
| WFS fulfillment (if used) | ~$3.45+ per unit (similar to FBA) |
| Shipping | Seller-arranged or WFS |
| **Effective fee on $15 item (most categories)** | $15 x 15% = **$2.25 (15.0%)** |
| **Effective fee on $30 item (most categories)** | $30 x 15% = **$4.50 (15.0%)** |
| **Effective fee on $15 electronics** | $15 x 8% = **$1.20 (8.0%)** |

Notes:
- No monthly subscription is a major advantage over Amazon Professional ($39.99/mo).
- Category breakdown: Electronics 8%, Health 8%, Home 10%, Apparel tiered (5%/10%/15%), Beauty tiered (8%/15%).
- New seller savings program available in 2026 -- reduced referral fees for qualifying new sellers.
- Lower traffic than Amazon/eBay, so sales velocity may be lower.

## SOURCE PLATFORMS (Where You Buy Inventory)

---

### Amazon (as a Source)

| Source Type | Typical Discount off Retail | Notes |
|---|---|---|
| Regular price | 0-10% | Use coupons, Subscribe & Save (5-15% off) |
| Warehouse Deals / Resale | 20-50% off | Open-box, returned items; condition varies |
| Renewed (refurbished) | 15-40% off | Certified refurbished with warranty |
| Outlet | 20-60% off | Overstock and clearance |
| Lightning Deals / Prime Day | 15-40% off | Time-limited; competitive |

Key consideration: Amazon-to-eBay arbitrage is the most common form. Buy at 30-50% off on Warehouse/Outlet, resell at near-retail on eBay/Mercari.

### Walmart.com Clearance

| Source Type | Typical Discount | Notes |
|---|---|---|
| Regular clearance | 25-75% off | In-store + online; seasonal cycles |
| Rollback pricing | 5-20% off | Temporary price reductions |
| Online-only clearance | 30-70% off | Often better than in-store |

Key consideration: Walmart clearance is excellent for retail arbitrage but inventory is inconsistent. Must scan frequently.

### AliExpress

| Factor | Detail |
|---|---|
| Typical product cost | $1-15 per unit |
| Shipping to US | $0-10 (many items have free shipping; 15-45 day delivery) |
| Minimum order | 1 unit (no minimum) |
| Typical resale markup | 2-4x cost |
| Quality risk | Moderate-high; must vet suppliers |

Key consideration: Shipping time (15-45 days) makes AliExpress unsuitable for drop-shipping where the buyer expects fast delivery. Best for pre-purchasing inventory. Chinese import tariffs in 2026 add 10-25% to COGS.

### Temu

| Factor | Detail |
|---|---|
| Typical product cost | $0.50-10 per unit (often cheaper than AliExpress) |
| Shipping to US | Usually included in price |
| Minimum order | 1 unit |
| Dropshipping support | **Not allowed** per TOS |
| Quality risk | High; very cheap goods |

Key consideration: Temu is NOT a dropshipping source. You can buy and resell, but you must receive and reship inventory yourself. Prices are extremely low but quality is inconsistent. Best for testing product viability.

## FEE COMPARISON SUMMARY

### Effective Platform Fee % (excluding shipping, on item price only)

| Platform | $15 Item | $30 Item | Notes |
|---|---|---|---|
| **FB Marketplace (local)** | 0.0% | 0.0% | Best: zero fees |
| **TikTok Shop (new seller)** | 2.8% | 2.8% | First 90 days only |
| **TikTok Shop** | 7.0% | 7.0% | Lowest ongoing fee |
| **Walmart (electronics)** | 8.0% | 8.0% | Category-dependent |
| **Mercari** | 10.0% | 10.0% | Simple flat rate |
| **FB Marketplace (shipped)** | 10.0% | 10.0% | Same as Mercari |
| **Etsy** | 12.5% | 11.0% | Fixed costs hurt small items |
| **eBay (no store)** | 15.9% | 14.6% | Highest mainstream fee |
| **Amazon FBM (Pro)** | 15.0% | 15.0% | Most categories |
| **Poshmark** | 20.0% | 20.0% | Highest overall, but shipping included |
| **Amazon FBA** | ~38% | ~27% | FBA fees crush low-price items |

### Ranking for Arbitrage (Best to Worst for Low-Margin Reselling)

1. **TikTok Shop** -- 7% all-in, growing audience, new seller promo
2. **FB Marketplace (shipped)** -- 10%, huge audience, zero for local
3. **Mercari** -- 10%, simple, good for general merchandise
4. **Etsy** -- 11-12.5%, but limited to niche categories
5. **eBay** -- 14.6-15.9%, but largest resale audience and trust
6. **Walmart Marketplace** -- 8-15%, lower traffic
7. **Amazon** -- 15%+ and $39.99/mo, only worth it at scale
8. **Poshmark** -- 20%, only for fashion/apparel with high margins
