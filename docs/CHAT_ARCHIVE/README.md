# 📂 Chat Archive

This folder contains saved chat sessions from GitHub Copilot and other AI assistants used during the EmpireBox project development.

## 📋 Purpose

Copilot Chat sessions don't sync across devices or browsers. This archive ensures important conversations, decisions, and context are preserved for future reference.

## 🗂️ File Naming Convention

Use the following format for archive files:

```
YYYY-MM-DD_topic_summary.md
```

**Examples:**
- `2026-02-19_initial_setup_decisions.md`
- `2026-02-19_stripe_integration_discussion.md`
- `2026-02-20_marketforge_architecture.md`

## 📝 What to Include in Each Archive

Each session summary should contain:

1. **Date & Session Info**
   - Date of conversation
   - AI model used (if known)
   - Approximate duration/length

2. **Key Topics Discussed**
   - Main subjects covered
   - Questions asked

3. **Decisions Made**
   - Any architectural decisions
   - Product decisions
   - Technical choices

4. **Action Items Generated**
   - Tasks identified
   - Next steps agreed upon

5. **Important Context**
   - Any context that would be useful for future chats
   - Links to related issues/PRs

## 📤 How to Save a Chat Session

### Option 1: Ask Copilot for Summary
At the end of a chat session, ask:
> "Summarize this session for my chat archive in markdown format"

Then save the output to a new file in this folder.

### Option 2: Manual Export
1. Copy the relevant parts of the conversation
2. Format as markdown
3. Save with the naming convention above

### Option 3: Comment on Issue #20
For quick saves, add a summary comment to Issue #20 instead of creating a file.
