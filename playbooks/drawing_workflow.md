# Drawing Workflow Playbook
1. Customer requests → identify item type + style from 204 options
2. Collect dimensions (W×D×H minimum)
3. Collect fabric/finish/hardware preferences
4. Generate PRESENTATION drawing (client-friendly)
5. Review with customer → revise if needed
6. Generate SHOP drawing (for fabrication team)
7. For WoodCraft items: generate CONSTRUCTION drawing (exploded + parts list)
8. Attach all drawings to job record
