# Starter Build (~$299)

Best for solo sellers or developers evaluating EmpireBox.

## Specifications

| Item | Detail |
|------|--------|
| **CPU** | Intel N100 (4-core, 3.4 GHz Turbo) |
| **RAM** | 8 GB DDR4 |
| **Storage** | 256 GB M.2 SATA SSD (internal) |
| **External Storage** | Samsung T7 500 GB USB 3.2 |
| **OS** | Ubuntu 24.04 LTS |
| **Network** | Gigabit Ethernet + Wi-Fi 5 |
| **USB** | 4× USB-A, 1× USB-C |
| **Power** | 15 W TDP |

## Supported Forge Products

At 8 GB RAM you can comfortably run 3–4 Forge products simultaneously:

- ✅ MarketForge
- ✅ ShipForge
- ✅ SupportForge
- ✅ ForgeCRM
- ⚠️ SocialForge (limited AI features)
- ❌ Ollama local AI (insufficient RAM)
