# MarketF Overview

## What is MarketF?

MarketF is EmpireBox's own peer-to-peer marketplace platform - an alternative to eBay with significantly lower fees (8% vs 12.9%). It's designed specifically for resellers who want to maximize their profits while providing a seamless buying and selling experience.

## Key Features

### 🔥 Only 8% Marketplace Fees
- **MarketF**: 8% marketplace fee
- **eBay**: 12.9% final value fee
- **Savings**: $4.90 per $100 sale

### 📦 Built-in Shipping via ShipForge
- Integrated shipping label creation
- Pre-filled address and package details from orders
- Automatic tracking sync
- Competitive shipping rates

### 🔒 Secure Escrow Payments
- Buyer's payment held in escrow until delivery
- 48-hour dispute window after delivery
- Automatic release to seller
- Stripe-powered payments

### ⭐ Verified Seller Ratings
- Buyer and seller reviews
- Transparent rating system
- Build trust with customers

## How It Works

### For Buyers
1. **Browse & Search** - Find products across multiple categories
2. **Purchase** - Secure checkout with credit card
3. **Track** - Monitor shipment status
4. **Review** - Leave feedback after delivery

### For Sellers
1. **List Products** - Create listings (or cross-post from MarketForge)
2. **Receive Orders** - Get notified when items sell
3. **Ship** - Use ShipForge integration for easy shipping
4. **Get Paid** - Receive payout after successful delivery
