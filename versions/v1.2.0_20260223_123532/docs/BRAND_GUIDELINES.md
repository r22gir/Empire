# Empire Brand Guidelines

## Colors
- **Primary:** Gold (#FFD700)
- **Secondary:** Orange (#FF8C00)  
- **Background:** Dark Blue (#1a1a2e, #16213e, #0f3460)
- **Text:** White (#FFFFFF)
- **Accent:** Green (#00ff88) for success
- **Error:** Red (#ff4444)

## Typography
- **Headers:** Bold, gradient gold-orange
- **Body:** Clean sans-serif (Segoe UI, system fonts)
- **Code:** Monospace

## Logo
- Location: ~/Empire/assets/empire-icon.png
- Use on dark backgrounds
- Minimum size: 32x32px

## UI Principles
1. **Dark theme default** - Professional, modern
2. **Minimal clutter** - Space is premium
3. **Gold accents** - Luxury, premium feel
4. **Large touch targets** - Easy to click
5. **Collapsible menus** - Show only what's needed
6. **Status indicators** - Always show system health

## Voice & Tone
- Professional but approachable
- Confident, not arrogant
- Clear, concise language
- "Empire" = power, control, ownership

- 🏰 EmpireBox - The hardware/platform
- 📊 Founder Dashboard - Control center
- 🧠 MAX - AI Assistant
- 💎 LuxeForge - Premium marketplace
- 🔨 ContractorForge - Contractor platform
- 🛒 MarketForge - E-commerce
