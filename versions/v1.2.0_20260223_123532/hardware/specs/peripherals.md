## Notes

- Configure label printers with **CUPS** on Ubuntu for USB printers.
- Network/Wi-Fi label printers are recommended for multi-user setups.
- Thermal printers require no ink — only thermal label rolls.
- Match label size to printer: 4×6 inch (102×152 mm) is standard for shipping.
