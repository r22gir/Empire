## Notes

- Prefer **Intel N100** or better for running multiple Docker containers.
- **16 GB RAM minimum** recommended for local AI (Ollama).
- **32 GB RAM** required for 13B+ parameter models.
- Choose a model with **2× M.2 slots** if you want redundant storage.
- Verify Ubuntu 24.04 compatibility before purchasing.
