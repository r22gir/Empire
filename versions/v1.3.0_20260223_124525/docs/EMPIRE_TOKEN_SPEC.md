## Utility Functions

- **Subscription discounts** — 20% off any EmpireBox subscription when paying with $EMPIRE
- **Marketplace fee reduction** — 6% trading fee vs. 8% standard
- **Staking rewards** — Earn yield by locking $EMPIRE
- **Governance votes** — 1 token = 1 vote on platform decisions
- **Referral rewards** — Paid out in $EMPIRE for successful referrals
- **Premium feature unlocks** — Exclusive features gated by token holdings
- **NFT License purchases** — Use $EMPIRE to buy / renew NFT licenses at a 20% discount
