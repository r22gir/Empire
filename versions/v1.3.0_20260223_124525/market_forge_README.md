# Market Forge Mobile App Documentation

## Project Structure
- `src/`: Contains all the source code.
    - `components/`: Reusable components.
    - `screens/`: Different screens of the app.
    - `assets/`: Images, fonts, and other assets.
    - `services/`: API services and data fetching logic.

## Features
- User Authentication: Login and registration processes.
- Dashboard: Overview of user data and insights.
- Marketplace: Browse and purchase products.
- Notifications: Real-time updates and alerts.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/r22gir/Empire.git
   cd Empire
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
