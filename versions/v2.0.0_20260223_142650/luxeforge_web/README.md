# LuxeForge Web

Marketing site for LuxeForge — AI-powered software for custom workroom businesses.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/r22gir/Empire/tree/main/luxeforge_web)

## Tech Stack

- Next.js 15 (App Router, static export)
- TypeScript
- Tailwind CSS
- lucide-react

## Local Development

```bash
cd luxeforge_web
npm install
npm run dev
# open http://localhost:3002
```

## Pages

| Route | Description |
|---|---|
| `/` | Homepage — hero, stats, features, pricing teaser, CTA |
| `/pricing` | Pricing tiers (Solo $79, Pro $249, Enterprise $599) |
| `/features` | Feature deep-dives |
| `/contact` | Demo request form |

## Deploy to Vercel

1. Push this folder to GitHub
2. Click the **Deploy with Vercel** button above, or import from the Vercel dashboard
3. Set any optional environment variables from `.env.example`
4. Deploy!

## Environment Variables

See `.env.example` for optional configuration.
