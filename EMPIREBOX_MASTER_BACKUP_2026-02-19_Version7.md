## 💡 KEY DECISIONS MADE

### Product Architecture
- **MarketForge & SupportForge** = Can be standalone OR EmpireBox sub-products
- **ForgeCRM** = Freemium model (basic free, premium paid), MVP-first approach
- **OpenClaw** = Universal AI brain powering ALL products

### SocialForge Model
- **Semi-automatic**: Pre-fills everything, user clicks "Create" for each platform
- **Fully automatic**: Only where platform APIs/TOS allow

### LLCFactory Model (Hybrid)
- **Partner**: Northwest Registered Agents for registered agent + state filings
- **DIY**: EIN applications + document filling (shared with ApostApp)

### VA Disability App
- **Telehealth IS LEGAL** with proper compliance:
  - Provider licensed in veteran's state
  - HIPAA-compliant video platform
  - Informed consent documented
  - Thorough evaluation records
- **Name options**: VetHelp Assist, VeteranForge, NexusForge

### Stripe Status
- Regular Stripe subscription started
- **Stripe Connect needed** for user payouts (not yet set up)
- Website completion is #1 priority before Stripe Connect

### Document Filler
- Shared system between ApostApp and LLCFactory (makes sense!)

## 📱 CHAT PRESERVATION NOTES

### Why Chat Disappeared
- Different devices have separate sessions
- Pro+ vs Pro subscription mismatch
- Model-specific chats (Opus 4.5 vs Claude)
- Models don't share context/history

### Prevention
- Always use the SAME model for important chats
- Export/backup important chats manually
- Store key decisions in GitHub repository
- Create issues for major milestones
