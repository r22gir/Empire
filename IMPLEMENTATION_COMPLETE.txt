================================================================================
                        USAGE EXAMPLES
================================================================================

BASIC USAGE:
```python
from empire_box_agents import TokenManager, RequestRouter

token_mgr = TokenManager(user_id="user123", tier="pro")
router = RequestRouter(token_manager=token_mgr)

# Simple task - uses local Ollama (free)
result = router.route_request(
    task_type="categorize_product",
    prompt="Blue Nike shoes"
)

# Complex task - uses cloud API (paid)
result = router.route_request(
    task_type="content_generation",
    prompt="Write product description",
    complexity="high"
)
```

BASE AGENT USAGE:
```python
from empire_box_agents import BaseAgent

agent = BaseAgent(user_id="user456", tier="empire")
result = agent.run_inference(
    task_type="categorize_product",
    prompt="Blue Nike shoes"
)
```

CUSTOM AGENT:
```python
from empire_box_agents import BaseAgent

class ProductAgent(BaseAgent):
    def categorize(self, description):
        return self.run_inference(
            task_type="categorize_product",
            prompt=f"Categorize: {description}",
            complexity="low"
        )

agent = ProductAgent("user789", "pro")
result = agent.categorize("Vintage watch")
```
